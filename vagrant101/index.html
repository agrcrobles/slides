Why Vagrant?

  • Project dependencies:
    • Databases
    • Programming languages
    • Development tools
  • Isolation Easy to use
  • Multiplatform
  • Compatible with multiple virtualization software:
    • VirtualBox / VMware / Parallels Snapshots

How it works

  • provision
  • ssh
  • suspend / resume
  • destroy
